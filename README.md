This project is a menu-driven Python program that performs various Digital Logic and Number System operations.
It is designed for students learning Digital Electronics, Computer Organization, and Python programming.
The program allows users to select an operation from a menu and performs the required conversion or logic operation interactively.

# Number System Conversions

Binary → Decimal

Decimal → Binary

Binary → Octal

Octal → Binary

Hexadecimal → Decimal

Decimal → Hexadecimal

# Digital Logic Gates
